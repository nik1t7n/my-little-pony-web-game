# Pony Korean Language Game

An interactive Korean language learning game built with React and TailwindCSS, where players guide a pony student through their first day at school while completing grammar exercises, vocabulary tasks, and cultural activities.

## Tech Stack

- React (Vite)
- TailwindCSS v4
- React Context API

## Getting Started

```bash
npm install
npm run dev
```

## Assets

- Background scenes and characters generated by **Nano Banana Pro**
- Pony emotion sprites cropped using [lazysticker.com](https://www.lazysticker.com/)

## Game Flow

1. School Entrance - Building selection
2. Hallway - Greeting practice
3. Timetable - Schedule overview
4. Korean Class - Grammar matching
5. Geography Class - Country identification
6. Cafeteria - Budget management
7. Class Activity - Date writing
8. Career Class - Vocabulary and grammar
9. Success - Celebration screen
